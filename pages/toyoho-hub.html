+ 关于远野妖怪保护区
岩手县远野市周围的北上山地。在深陷山中的地方，远野妖怪保护区就在那里。区内居住、栖息着一般被称为"妖怪"的异形，也有着很少与人类之间诞生的"半妖"生活着。保护区所在的异界原本被称为"寒户乡"，自古以来就保持着由山神和土著民组成的共同体系。区内充满了神秘的自然风光，例如神宿之灵峰，狂暴之山脉和无底之川渊，可以在四季中欣赏各种各样神秘的风景。村落里还排列着纯正和风的建筑，同时妖怪们所拥有的独特文化也在此扎根。

寒户乡作为近代自治体的历史，可以追溯到太平洋战争的末期。明治以后，常态维持机构对怪异的镇压持续加剧，各地的妖怪集团不得不解散和潜伏。曾经有一段时间，他们被许诺通过与军队合作并成为“妖怪大队”的一员而受到一定程度的优待，但是当帝国的败相日趋显露，妖怪们就预见到了未来可能会失去后盾，于是急于确保一个生存圈。1945年8月上旬，妖怪营的士兵们愤然起义，之后毅然向寒户乡集体移民，这次迁徙后来被称为“百鬼夜行”。妖怪们宣布设立“远野妖怪独立区”，并巩固了村落的防备。

战后，独立区更名为保护区，正式引入行政职能，例如成立政府机关和改善交通网。为了警戒外部的袭击，该区长期实行半锁国政策。但是，与外界隔绝会引起大大小小的问题，保护区逐渐被迫改变其政策。1989年，以外界进入新时代[[footnote]] 平成时代 [[/footnote]]为契机，保护区逐渐开放，开始接受新兴怪异，并开展面向超常市民的观光旅游事业。妖怪保护区现在以"朴素、神秘、残留着衰落时代气息的城市"为口号，提出了多种多样的地域振兴策略。

+ 寒户

= [[image http://scp-jp-storage.wikidot.com/local--files/file:3266034-182-wfol/horoyoi.jpg width="400px"]]

= [[size smaller]]黄昏时的微醺巷。 雾状酒开始升起。[[/size]]
@@@@
寒户是人类、半妖、都市妖怪生活的地区。是从“百鬼夜行”以前就开始繁盛的传统村落，大致分为上寒户和下寒户。上寒户是妖怪保护区的中心，除了以政府机关为首的行政机关以外，还有很多商家和娱乐场所。与上寒户形成鲜明对比的是，下寒户保留着很多朴素的田园和日式曲屋，这些日本的原始风光深深吸引着游客的目光。

++ 上寒户

* 人间町
[[div style="margin: -0.5em 0 1.5em 3em;"]]
@@@@
被护城河和石墙包围着，有种城下町的氛围。根据传说，是一个外界的妖怪木匠仿照远野南部氏[[footnote]]八户氏[[/footnote]]的城堡修建，甚至还建造了一座模仿锅仓城的小城堡。但是对于普通妖怪的生活好像有点不方便，城堡现在变成了付丧神的住处。
[[/div]]


* 微醺巷
[[div style="margin: -0.5em 0 1.5em 3em;"]]
@@@@
由从[[[groups-of-interest-jp#drunkness_street|酩酊街]]]回来的人所经营的一条区内首屈一指的欢街。因为被酩酊街的区域反复侵蚀和后退，即使是大白天巷内也很昏暗，夜晚则是雪花飞舞。如果在这里忘记了什么东西的话，还是放弃寻找比较好。没关系，因为您很快就会忘记自己忘记了什么的事情。
[[/div]]


++ 下寒户

* 岩手县警远野警察署 寒户驻在所
[[div style="margin: -0.5em 0 1.5em 3em;"]]
@@@@
根据与日本政府达成的协议，派驻了人类的警察。严格说来，它不是一个旅游景点，但似乎“守护妖怪的巡警”很难在外面看到，它正成为为寻求情感和喜好拍摄的游客之间的隐藏名胜。
[[/div]]


* 福屋敷
[[div style="margin: -0.5em 0 1.5em 3em;"]]
@@@@
这是福神座敷童子居住的建筑。因在别人家里走来走去，因给房东带来幸福但其离开时会发生可怕的灾难而闻名的座敷童子。受灾害困扰的村民在某一年，一点点拿出自己的财产，建立了座敷童子专用的空房子，成功地阻止了这种不幸的搬家。现在作为可以和神轻松地玩的场所被大家所喜爱。
[[/div]]


* 迷家关[[footnote]]译注：“迷家”指日本东北至关东地区流传的传说，会给迷路进去的人带来财富。[[/footnote]]
[[div style="margin: -0.5em 0 1.5em 3em;"]]
@@@@
带有一座美丽的红白色花园和气派大门的曲屋。发挥着保护区与外界之间的关卡的作用，并暂时滞留着那些从外界入乡的人。内部待遇因入乡者的经济状况而异，为穷人提供豪华服务，为富人提供极其恶劣的服务。
[[/div]]


+ 新寒户

= [[image http://scp-jp-storage.wikidot.com/local--files/file:3266034-182-tu05/Apaato.jpg width="400px"]]

= [[size smaller]]新寒户的地标“杉泽公寓”。是一栋带浴室和厕所的先进住宅。[[/size]]
@@@@
新寒户开拓在西北偏僻地区，安置了移居远野的新兴怪异……成为俗称的“都市传说”的居住区。用混凝土建造的现代建筑非常引人注目，在寒户乡的老居民们之间有着“寒户乡的纽约”等外号。与在保护区生活了很久的传统妖怪不同，新兴妖怪对外界的人不太感冒，所以到访时需要注意冷淡的态度、突然的鬼压床、轻微的诅咒等情况。另外，新寒户还设有学习外界文化和技术的设施“现学讲习所”，是保护区近代化事业的枢纽。

+ 鹿仓山

= [[image http://scp-jp-storage.wikidot.com/local--files/file:3266034-181-hnf2/yama.jpg width="400px"]]

= [[size smaller]]从熔岩流中眺望鹿仓山。[[/size]]
@@@@
被称为“寒户富士”的圆锥形成层火山。这是一座有着山神坐镇的灵峰，山女和山姥居住在那里并充当着神的从者。寒户的山神是被怪异信仰的罕见神格存在，其奇魂是支撑寒户乡空间的灵力源泉。在民间传说中，在老薮椿树[[footnote]]译注：“椿”是日本对山茶花的称呼。[[/footnote]]中寄居的灵在漫长的岁月中升华为神，民俗学者指出“山神也是以某种妖怪为根源的”。
@@@@
@@@@
* 鹿仓山神社
[[div style="margin: -0.5em 0 1.5em 3em;"]]
@@@@
在祭祀山神的神社里，巨大的薮椿树作为御神木坐镇着。每年春天都会举行神降仪式，可以看到神木发光，山神显现的神秘景象。
[[/div]]


* 寒户乡妖怪保护中心
[[div style="margin: -0.5em 0 1.5em 3em;"]]
@@@@
是管理住在山里、森林、等偏僻地方的妖怪的大规模设施。虽然他们看起来是不会说话的怪物，但他们大多具有正儿八经的人格，拥有与常人一样的思考能力。在参观的时候请不要疏忽大意。
[[/div]]


* 蒐集院资料馆
[[div style="margin: -0.5em 0 1.5em 3em;"]]
@@@@
收藏着与蒐集院有关的各种物品和妖怪相关的资料。蒐集院的资产在战后大半被基金会接收，因此很难追溯他们的真实形象。本馆是以研究之前的蒐集院、以超常市民为基础的国内首屈一指的设施，并由前成员及其子女运营。馆长由日奉薮区长兼任，而且在现代也继承了“研仪官”的头衔。
[[/div]]


+ 隐岳

= [[image http://scp-jp-storage.wikidot.com/local--files/file:3266034-182-5buu/fox.jpg width="400px"]]

= [[size smaller]]隐岳因为经常发生狐的出嫁而闻名。[[/size]]
@@@@
穿过鹿仓山山脚，前面显眼的山岳地带耸立着一片陡峭的岩壁。因为危险的地方非常多，加上急剧变化的气候，所以这里以遇难人数多而出名。然而也有着眼于这种环境的人，例如犯下罪过的妖怪、山贼、仙人等，有一两个怪癖的人们把这里作为临时的居所。另外，隐岳自古以来就被称为“[[span class="ruby"]]御岳[[span class="rt"]]おぬがたけ[[/span]][[/span]]”，从它的发音来看，传说可能和妖怪中的鬼有某种关系。

* 雷暴发电厂
[[div style="margin: -0.5em 0 1.5em 3em;"]]
@@@@
作为现代化项目的一部分而建造的灵力式发电厂。由于发电量取决于雷神的心情，因此设施工作人员每天都会进行感谢登山。
[[/div]]


+ 丹德拉野[[footnote]]译注：柳田国男记载于《远野物语》中的地名，是超过60岁的老人们移居于此，边劳动，边等待死亡之地。不少人认为其是“蓮台野”的东北方言的讹变。[[/footnote]]
是隐岳西部广阔的未开发原野。其中一部分是如月工务店的用地，建有研修设施和资材仓库等。不推荐游客访问。

+ 伊哈托夫合区[[footnote]] 译注：伊哈托夫，来自宫泽贤治童话的地名[[/footnote]]

> = 每當路遇險阻 四次元藝術的龐大和深度也將隨之增加
> = 詩人把他的痛也當做享樂
> = 永不完成之物也即是完成[[/size]]
> [[>]]
> ―― 宫泽贤治[[footnote]] 译注：宫泽贤治，现实存在的人物，日本昭和时代早期的诗人、童话作家。[[/footnote]]『农民艺术概论纲要』结论[[/>]]
@@@@
= [[image http://scp-jp-storage.wikidot.com/local--files/file:3266034-186-h919/ihatobu.jpg width="400px"]]

= [[size smaller]]某日的草原。同样的景色再也见不到了。[[/size]]
@@@@
以已故宫泽贤治先生的心象世界为依据的区域。北边是雄伟的山地，南部的平原是广阔的农田和牧草地，中央有一个叫莫利奥的小镇。原本伊哈托夫与寒户乡位于不同的空间，但在1933年宫泽氏去世后，伊哈托夫的形态崩坏时触碰到了寒户乡，并与这块土地融合在了一起。合区最大的特征，是频繁发生被称为“凯谷”的异常现象，当人突然注意到的时候周围的景色完全变了……这样的事情并不少见。这一现象也适用于生物，但是游客作为“德库沙桑”会受到一定程度的保护，所以活动相对比较安全。
@@@@
@@@@
* 区役所伊哈托夫分所
[[div style="margin: -0.5em 0 1.5em 3em;"]]
@@@@
建在与鹿仓山接壤的凯谷较少发生的地面上的分政府建筑物。紧挨着宿场街，作为探索伊哈托夫的前哨地发挥着作用。
[[/div]]


* 银河铁道站
[[div style="margin: -0.5em 0 1.5em 3em;"]]
@@@@
位于莫里奥市[[footnote]] 译注：莫里奥市，来自宫泽贤治童话的地名。应该是世界语盛冈市的音译。[[/footnote]]的新银河铁路的停车站。驶向夜空的机车的雄姿真是一幅壮丽的景观。来自外界的宇宙，或者是来自空想的旅客都在这里停留，营造出了一种不可思议且国际化的氛围。
[[/div]]

* 安布罗斯・山猫轩
[[div style="margin: -0.5em 0 1.5em 3em;"]]
@@@@
由外界著名的餐厅连锁店・[[[ambrose-restaurant-hub|安布罗斯]]]经营的西餐厅。大量使用形而上的食材，为你带来美好的叙事体验。
[[/div]]


+ 河童渊

= [[image http://scp-jp-storage.wikidot.com/local--files/file:3266034-182-7cij/mati.jpg width="400px"]]

= [[size smaller]]在深渊的彼方耸立着的什么的影子。没有人到达，在当地被认为是蜃气楼[[footnote]] 译注：蜃气楼，中国和日本传说中的妖怪，会吐气形成海市蜃楼。形似大牡蛎（一说是水龙）。[[/footnote]]所为。[[/size]]
@@@@
居住着以河童为首的水生妖怪。与外界的河童渊[[footnote]]译注：河童渊，传说存在于现实远野市的一条小河中的深渊，是河童的家乡。[[/footnote]]相连，可以通过特定的手段出入。寒户侧的河童渊被形容为“水海”，广阔得可以把保护区的陆地团团围住。在海上有为了保护空间而设的结界和“百鬼夜行”后设置的炮台等，以防备外敌的入侵。居民大部分住在沿海地区，另一方面也有像大名主[[footnote]] 译注：大名主，日本传说中的妖怪，一条通过摇晃身体引起地震的大鲶鱼。[[/footnote]]一样生活在渊底的人。
@@@@
@@@@
* [[[surface-to-air-bamboo|竹枪防空林]]]
[[div style="margin: -0.5em 0 1.5em 3em;"]]
@@@@
将在与调查局的战斗中缴获的[[[SCP-518-JP|对空爆竹]]]进行品种改良，并种植在沿岸一带。受到了山的灵力增幅而被大幅度强化，多次击落了穿过结界的侵入物。与不定期飞来的式神进行缠斗的空中战也被称为“河童渊的水上烟花”，为寒户的夜空增添了色彩。
[[/div]]

介绍在妖怪保护区活跃的人，移居的人，和相关的人。

+ 日奉薮 / 区长
远野妖怪保护区的第一代区长，鹿仓山神社的潮来长[[footnote]] 译注：潮来，是日本东北地区恐山的一种可以呼唤死者让死者附身在自己身上的巫女。[[/footnote]]，人类。生于日奉家的旁流。为了寻求安稳的生活和神格研究者应有的状态，而移居寒户乡。作为山神的巫女而工作，一直致力于空间的维护。在保护区成立时，以被当地人民推举的形式就任区长。从那以后就致力于保护区的行政工作。特长是改变容貌，所以不在区长室的话就不知道是谁。利用特长的忍者技能进行每日的视察工作。

+ 站长桑
银河铁道站的站长。有着三色毛的猫又，以前经营车站附近的面向猫的事务所[[footnote]] 译注：《猫的事务所》，宫泽贤治所著童话。[[/footnote]]。因为对合区的地势很熟悉，附近有一条粗管，作为该区的意见番活跃着。对于食物很贪婪，经常能看到在区内各处寻找珍馐美味的身影。
+ 寒户婆
寒户一带的实质土地所有者，作为名士从事地区内的调停等工作。在婆婆所在的地方经常刮风，虽然给周围的人一种威吓的感觉，但她有时却有一个好玩的一面，例如头发被吹蓬乱而发牢骚等。
+ 白狐桑
2020年刚搬来的[[[gumiho|妖狐]]]。原本是人类，但因为被[[[searching-for-the-demon-foxes-1|帝国异常事例调查局(IJAMEA)]]]进行了人体实验，变成了拥有九尾的白狐。现在住在鹿仓山的妖怪保护中心，在蛇手和调查局OB的帮助下，为了回归社会进行康复训练。

+ 笼守桑
警防团中的重要人物。负责管理河童渊的防空林，并且是区内最擅长使用竹子的人。此外，区内燃放的竹烟花也是笼守桑的作品。

+ 松泽铃桑
白狐的监护人和主要照顾者。下半身是蛇的后天半妖，隶属于警防团・退役妖人会。平时在防空林做烧炭的工作，把多余的竹子砍掉然后做成竹炭供应给区里。妖怪营的原成员，在移居之前在[[[ryogen|遥远的地方担任炎神的巫女。]]]

+ 玛丽桑
从东京移居到寒户乡，对电波十分熟悉的新兴怪异。受保护区之邀担任IT推进特命顾问，正在为寒户的数字化而奋斗。是为数不多[[[atashi-merisa|遭遇了现代基金会职员]]]的移居者，其恐怖体验让听者为之震惊。

+ S・Y桑（化名）
来自外界的巡警，因为[[[kanshin-ryoiki-tono|协议]]]的原因，详细情况无法说明，但确实是个非常好的人。

+ 名誉居民

++ 佐佐木喜善先生[[footnote]] 译注：佐佐木喜善，现实存在的人物，日本民俗学者、作家。[[/footnote]]
民俗学家、资料收集家。远野・土渊出身。误入寒户乡，成功地和当地妖怪加深了友谊。出版了很多关于妖怪的著作，为外界对妖怪的了解做出了巨大贡献。

++ 柳田国男先生[[footnote]] 译注：柳田国男，现实存在的人物，日本民俗学创立者、官员。也即是《远野物语》的作者。[[/footnote]]
民俗学家、官员。[[[collected-item-no0724|蒐集院关东傭人连总代]]]。在寒户乡进行广泛的调查的同时，与常态维持机关进行了顽强的交涉，确立了战后寒户乡的存续。之后担任政府的枢密顾问官，致力于维护国内超常组织。

++ 宫泽贤治先生
诗人、童话作家。 生于花卷。 在佛教和农民文化的基础上进行创作，创造了心中的理想乡—伊哈托夫。晚年与佐佐木建立了交流关系，这被认为是伊哈托夫出现在寒户乡的主要原因。

+ 超常民俗研究家 佐藤某氏所整理保护区成立的简史

= [[image http://scp-jp-storage.wikidot.com/local--files/file:3266034-184-5kwv/%E4%B8%80%E3%81%A4%E7%9B%AE%E7%8E%89.png width="100px"]]

= [[size smaller]]寒户的妖怪家族中使用的家纹“一之大目玉”。在区旗的设计中加入了其元素。[[/size]]

异界「寒户乡」是远野「山和森林的神秘」具象化而成的山神的神域，在东北山区中无论从哪里都有可能神隐而闯入的空间。当时的寒户，除了山姥[[footnote]]译注：山姥，日本传说中的妖怪，化身成为美丽的妇人，提供在山中的旅人住宿和食物，但是，在旅人就寝后，将其吞食。[[/footnote]]、经立[[footnote]]译注：经立，日本东北地区传说中的妖怪，一般认为是猴子精或者鸡精，会袭击村子掳走女性。[[/footnote]] 等远野妖怪外，还有后来加入的[http://scp-wiki-cn.wikidot.com/groups-of-interest-jp 如月工务店]的鬼人众，形成了一个松散的团体。就这样异界的存在以一种传承的方式扎根于该地区，并在之后被民俗学家佐佐木喜善“发现”。他希望与山神和远野妖怪结成友谊，这一点被接受了。

20世纪初，文明开始进入这个和平的异界。佐佐木和寒户的土著民们为了逃离[[[ijamea-hub|帝国异常事例调查局]]]主导的“白铎计划”的魔爪，并竭力防止他们的存在暴露在阳光下。佐佐木喜善的朋友柳田国男在这方面发挥了重要作用。虽然他不是一位术士，但在[http://scp-wiki-cn.wikidot.com/groups-of-interest-jp#toc3 蒐集院]中有着一定地位，经过努力总算是蒙蔽了调查局的目光。《远野物语》[[footnote]]译注：《远野物语》，现实中的著作，柳田国男在1910年（明治43年）发表的整理岩手县远野市出身的佐佐木喜善所说的远野民话而成的传说集。日本民俗学的名著。[[/footnote]]也被认为是当时对策的一环。之后，调查局的兴趣开始转向朝鲜满洲，而处于日本偏僻乡下的远野也不再被关注。

= [[image http://scp-jp-storage.wikidot.com/local--files/file:3266034-182-hw0v/miko.jpg width="300px"]]

= [[size smaller]]大正时代的日奉薮区长。（蒐集院资料馆蔵）[[/size]]

第二次转机发生在第一次世界大战前不久。在柳田国男的帮助下，人类移居者来到了寒户乡。当时有一名作为巫女的能力非常强的女术士，而且她和山神的协调程度非常惊人。因为被立为神社的巫女，所以立刻稳定了寒户乡的空间。那就是背叛了神殺的一族・[http://scp-wiki-cn.wikidot.com/system:page-tags/tag/%E6%97%A5%E5%A5%89%E4%B8%80%E6%97%8F 日奉家]的一员，被追捕的[[[kanshin-ryoiki-tono|日奉薮]]]。刚入乡的时候，大家都很讨厌她，但是不久就被居民们接受了。佐佐木喜善、柳田国男、日奉薮，还有十多年后才与寒户乡接触的[[[poi-1933|宫泽贤治]]]，正因为有这四个人的存在，才有了远野妖怪对人类的理解。

1933年9月末，寒户乡失去了佐佐木喜善和宫泽贤治。他们是寒户的好朋友，为寒户留下了很多东西。最具代表性的例子就是另一个被称为“[[[dreamlandiwate|伊哈托夫]]]”的异空间。为了稳定与寒户乡相邻而出现的这个极不稳定的异界，日奉薮付出了巨大的努力。伊哈托夫与寒户乡的文化有很大的不同，让妖怪们感受到了人类社会，因此乡民们的反应也各不相同。

1945年7月中旬，第三次转机到来。妖怪大队的队员向寒户乡提出了集体移居的申请。当时，被送到战场的妖怪们背叛了调查局和祖国日本，并正在寻找着新的家园。虽然设想了各种各样的问题，但日奉薮决意接受他们，柳田国男也表示赞同，所以原住民们也同意了这一请求。

妖怪大队的意见也不统一，300名左右的成员中，约有200名决定起义和移居。再加上当时冲绳的战况，在最前线战斗的他们很难渡过没有制海权的近海。在[[[rough-sea|判官会]]]的协助下，通过潜入油船和潜水艇等危险手段，成功到达本土的只有原人数的一半约100人左右。

= [[image http://scp-jp-storage.wikidot.com/local--files/file:3266034-183-cq89/train.jpg width="300px"]]

= [[size smaller]]在东进途中，银河铁道遭到调查局的高射炮攻击。[[/size]]

寒户乡为他们提供了宫泽贤治的遗产之一，“银河铁道[[footnote]] 译注：《银河铁路之夜》，宫泽贤治所著童话。[[/footnote]]”。通过如月工务店和[http://scp-wiki-cn.wikidot.com/groups-of-interest-jp#toc2 东敝钟表电机]进行的修理，这辆机车总算恢复了在三维空间飞行的功能。因此，8月初“百鬼夜行”被强行执行，在途受到了调查局、蒐集院、[[[moai-tale-02|五行结社]]]等的阻挠，虽然出现了逃亡者，但百鬼中大约有6成到7成的人成功移居到了寒户乡。

“百鬼夜行”实在太过引人注意，所以调查局和蒐集院知晓了寒户乡的存在，因此不得不明确对外方针。为了对抗接二连三的五行结社的袭击，“远野妖怪独立区”这个名称就是在这个时候产生的。

随着战争的结束，日本国内的超常组织遭到了重组的浪潮。蒐集院和调查局都陷入了无法顾及远野这种弱小群体的境地。利用这种情况，远野踏实地做好了防备。但是，当时反对蒐集院姿态的一派术士加入/移居远野。因此，蒐集院也开始采取应对远野的措施。8月末，蒐集官・波户崎恺被派去与独立区交涉。根据交换的协议，远野被定义为“保护”妖怪的场所。这就是“远野妖怪保护区”成立的大致情况。

蒐集院决定不干涉远野，其消极政策也被基金会继承。在与蒐集院断绝关系成为枢密院顾问官的柳田国男的庇护下，远野妖怪保护区一边收容移民，一开始了一段安宁的时光。

++++ 百鬼夜行
1945年8月上旬进行的妖怪们的集体逃亡行动。被五行结社等势力盯上，在到达远野之前出现了不少掉队者。

++++ 狐的出嫁
> ・在保护区北部的隐岳山地，频繁发生异常气候现象
> ・其中一种气候，就是女狐为了寻找结婚对象而降临导致的“狐的出嫁”
> ・幸运的在地面上的男人，有时会被狐狸相中，并与之结合
> ・虽然不问男人的种族，但因为狐狸也很挑剔，所以也有可能没找到喜欢的人就结束了

隐岳发生的异常气象现象之一。
出嫁的狐狸和吃人/化人的狐狸是不同的种族，和在异类婚姻谭[[footnote]] 译注：异类婚姻谭，人类和不同种族的存在结婚的一类传说的总称。[[/footnote]]中出现的“为人而化”的狐狸很接近。

+ 社会・文化
++++ 足
> 「这里是下寒户村。我马上叫百足过来，请等一下。」
>
> 当我在阳光下晒了约10分钟太阳时，我看到一个红色的轮廓以爆炸性的速度从马路的另一边飞来。
> 仔细一看我发现那东西不知为啥扭来扭去的……
>
> 那是一条公交车大小的大蜈蚣。（[[[news4pamwac-fox|狐娘]]]）

作为区内的交通工具之一，外观是汽车大小的百足。是三上山大百足[[footnote]] 译注：来自平安时代中期武将藤原秀乡在三上山退治大百足的传说。[[/footnote]]的后裔，被居民当作出租车使用。坐着舒服的体节是其特征。

++++ 肉糜
> 「给那个肉块喂食有点恶心。那个真的没问题吗？」
> 「不管怎么说，如果不定期刮掉碎肉块的话，野箆坊会因为代谢综合症而死的。」（[[[dialogue-at-toono|曲松]]]）

把野箆坊[[footnote]] 译注：野箆坊，江户时代的妖怪画卷中的妖怪。外观就好像一坨肉丸上长了手脚。中文也叫肉人或者肉瘤怪，没有确定的中文译名，野箆坊和肉瘤怪是否是同种妖怪尚有争议。[[/footnote]]多余的肉割掉，加工而成的一种食物。味道接近人肉，作为食人妖怪的代替食品被视为珍宝。战后一段时间几乎被当作主食消费，拯救了粮食困难的妖怪保护区。

++++ 夜行祭
> 下午6点，依靠狐火的照耀，约800名区民开始登山祈祷。他们载歌载舞，步行至位于山腰的鹿仓神社。在神社，山女山姥等人举行了降神仪式。在日奉区长参拜后，古山茶花的神木开始发光，以山神降临至区长身上。（[[[toyoho-202004|广报20年4月号]]]）

春天举办的节日。山神显现，并举行视察和宴会等活动。

+ 行政・组织
++++ [https://ja.wikipedia.org/wiki/%E8%AD%A6%E9%98%B2%E5%9B%A3 警防团]
> 为寻找犯人，警防团计划令无人机变为付丧神以获取情报。（[[[toyoho-202004|广报20年4月号]]]）

为了在敌人的袭击和灾害中保护区民而结成的团体。作为实质上的警察、消防部门发挥着作用。超过团员的能力和理解的东西有时会借助特事科的力量。

++++ 丰穗广报
区政府发行的宣传报。“丰穗「とよほ」”是“保护区「とおのようかいほごく」”的缩写。因为是面向区民编写的报纸，所以很多新词、外来语都被换成了区民能看懂的古语。

++++ 三丁社
[http://scp-wiki-cn.wikidot.com/groups-of-interest-jp#toc12 特里斯墨吉斯忒斯翻译与运输社]面向远野的公司名。由众神经营，承包保护区的现代化事业和贸易等。不懂英语的居民是完全记不住其正式名称的。

++++ 对外课
在外界从事各种活动的部门。公共关系、情报和庇护支持等，对保护区来说是一项难度很高的工作，除了人类以外，还有像妖狐这样善于隐秘行动的妖怪。

++++ 迷家
进入保护区的方式之一。位于岩手县一带的深山里，地处连兽道都没有的偏僻地区。也有的是常态维持机关故意设置的陷阱，进入时请务必小心。

+ 外界
++++ 外界/外部/区外
妖怪保护区之外的人间界。被丢弃在外界的山上的垃圾有时会转移到或漂到保护区内（因为山神的反复无常）。

++++ 新兴怪异
近现代发生的新的怪异。也被称为都市传说。因为和妖怪相比生存经验比较少，所以经常被恶势力所吸引，成为基金会・GOC探查保护区的绳索。对此区民们都抱着同情的心理，现在对外科正在进行救援行动。

+ 伊哈托夫
++++ 新银河铁道
> 不管怎么说，在新银河铁道将路线延伸到银河联邦成员国之前，很多妖怪在向远野妖怪保护区逃窜的途中落入了基金会和GOC的魔爪。特别重要的是因为不用通过逢坂关，所以不用和五行结社打交道。（[[[ambrose-wildcat-house|山猫轩]]]）

在第四幻想中前进的机车，是对因作者死亡而坠毁的银河铁道进行整修的产物。 进入保护区的公共交通方式之一。 最初仅在保护区内运作，但不久就延伸到了“银河联邦[[footnote]]译注：银河联邦，现实存在的组织，日本航空航天局在日本的市町村交流组织。[[/footnote]]”的“加盟国”，并开展运营了诸如“平泉净土线”之类的支线。 理论上列车是按照天空上的距离而不是银河系的实际距离运行。
++++ 咕啦姆嘣[[footnote]]译注：咕啦姆嘣，来自宫泽贤治短篇小说《山梨》的生造词汇。其具体表现了什么学界尚有争论。[[/footnote]]
> 在多数居民的自发协助下，伊哈托夫合区至少捕获有42只该实体。由于其具有反认知的异常性质，所有实体再次迅速地下落不明。尽管搜索行动很明显并不必要，但保护区仍然在投入预算进行搜索。（[[[kanshin-ryoiki-tono|特别关注区域]]]）

居住在伊哈托夫合区的生物实体。具有一定的反认知性质，具体情况尚不清楚。从对当地居民的询问中得到了“跳起来笑”、“死亡”、“被杀”这样的话。在区政府伊哈托夫分所，搜寻咕啦姆嘣是区役所伊哈托夫分所的任务之一。
